# Wins Log

Append-only log of shipped work, glue work, mentorship, and impact. One file per year (e.g. `2026.md`).

Format and philosophy: see `~/.ai-shared/references/wins-log.md`.

## Quick Format Reminder

```
2026-05-15 — Shipped feature X. Pre-launch error budget held at 0.3%.
2026-05-14 — Closed long-standing auth contract ambiguity with BE team.
```

One to three lines per entry. Date first. "Did X, with Y impact / for Z reason."

## Privacy

These entries are personal. If `~/.ai-shared` is ever shared or made public, consider gitignoring `wins/*.md` (keep this README tracked).
